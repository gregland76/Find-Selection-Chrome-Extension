# Find Selection Chrome Extension

Extension Chrome/Edge pour rechercher rapidement le texte sélectionné sur une page web via différents moteurs de recherche (Google, Bing, DuckDuckGo, Leboncoin).

## Fonctionnalités
- Sélectionnez du texte sur n'importe quelle page web
- Cliquez sur l'icône de l'extension pour ouvrir un menu
- Choisissez le moteur de recherche souhaité (Google, Bing, DuckDuckGo, Leboncoin)
- Ouvre la recherche dans un nouvel onglet
- Bouton "À propos" personnalisable
